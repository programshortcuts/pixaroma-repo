02-02 create venv in dir

1

My project is in ~/scrapy-freecodecamp

create venv and activate in dir,

python -m venv vev
source venv/bin/activate

2- install scrapy

you can get command from website or here https://docs.scrapy.org/en/latest/intro/install.html

pip install scrapy

Then run scrapy, and you should see list of commands

scrapy

3- Install requirement.txt

As seen at https://github.com/Comfy-Org/ComfyUI

Make sure you are in (comfyui) enviroment, and you are in the ComfyUI directory created in the last step

paste this command

pip install -r requirements.txt

I ran into this error, Make sure python and pip are running through the miniconda enviroment, I had alias for both pip and python

4- run comfyui

Run the main.py python script in the ComfyUI directory to get server

cd ComfyUI
python main.py
